# エクササイズ2

このエクササイズではコンフリクトの解消方法を学んでいきます。

> この行を変更して下さい！

このエクササイズは以下の手順に従って下さい。

1. このリポジトリをクローンして下さい(既にエクササイズ1でこのリポジトリをクローンしている場合はこの手順をスキップして下さい)

2. `danger` というブランチを作成し、そのブランチに移動して下さい

3. Atomなどのテキストエディタで「この行を変更して下さい！」という行を「Warning!」という文字列に変更して下さい

4. ファイルの変更をコミットメッセージと共にコミットして下さい

5. `master` ブランチに移動して下さい

6. 「この行を変更して下さい！」という行を、あなたの好きな映画の名ゼリフに変更して下さい

7. ファイルの変更をコミットメッセージと共にコミットして下さい

8. `git merge danger` コマンドで `danger` ブランチを `master` ブランチにマージして下さい

9. コンフリクトが発生しますが慌てないで下さい！

10. Atomなどのテキストエディタでコンフリクトを解消して下さい(`<<<<<<`や`======`、`>>>>>>`を削除するのを忘れないでください！)

11. コンフリクトを解消するための変更をコミットして下さい

12. `git status` コマンドで `danger` ブランチが `master` ブランチにマージされた事を確認して下さい
